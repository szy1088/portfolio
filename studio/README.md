Sanity Studio for the portfolio.

Steps to run locally:

1. Install Sanity CLI globally: npm install -g @sanity/cli
2. From this folder run: npm install
3. Configure NEXT_PUBLIC_SANITY_PROJECT_ID and NEXT_PUBLIC_SANITY_DATASET in .env
4. Start studio: npm run start

To connect to sanity.io create a project at https://www.sanity.io/manage and set projectId and dataset.
